# Interface: ReactPortal

Represents a JSX element.

Where [ReactNode](../type-aliases/ReactNode-1.md) represents everything that can be rendered, `ReactElement`
only represents JSX.

## Example

```tsx
const element: ReactElement = <div />;
```
